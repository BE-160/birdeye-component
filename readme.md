# Birdeye Components

## Packages

* Header
* Styles
* Button

## Setup
* `git clone <repo url>`
* `npm install`

## Publish Changes
* Run `lerna bootstrap`
* Run `lerna publish`

## How to publish new component
* First commit your new component with git repo using `git commit -am "msg"`
* Run `lerna bootstrap`
* Run `lerna publish`
